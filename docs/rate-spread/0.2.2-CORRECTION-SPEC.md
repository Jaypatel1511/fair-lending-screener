## Tier 2 — now VERIFIED; fold into 0.2.2

### "disparate impact analysis" is a mislabel — VERIFIED (promote to Tier 1)
An adjusted protected-class coefficient held net of legitimate controls is the **disparate-TREATMENT** statistical test, not disparate-impact. Verified across fair-lending legal/industry sources (July 14 2026): "a loan-approval regression that finds a significantly higher denial rate for minorities… controlling for credit characteristics, establishes a prima facie case for **disparate-treatment** discrimination"; disparate *impact* targets a facially-neutral policy's *effect* and does not control the mechanism away (controlling for it causes "included-variable bias" in impact tests). The package's own methodology body already frames the disparate-treatment question ("similarly situated applicants"), so the "disparate impact" **label contradicts the text.**
- **Fix:** replace "disparate impact analysis" everywhere with the neutral, accurate **"lending-disparity screening"** / **"adjusted denial-disparity analysis"** (the Tier-1 taglines already do this). Do NOT relabel to "disparate treatment" — that word imports intent, which a screening tool must not assert. If the methodology doc names a theory, say the method "aligns with the statistical approach used in disparate-treatment screening," never "detects disparate impact."
- Locations beyond the Tier-1 taglines: scan `docs/methodology.md`, `_methodology_doc.md`, README body, and the notebook for any remaining "disparate impact" and neutralize.
- Sources: [ncontracts — regression tests disparate treatment](https://www.ncontracts.com/nsight-blog/everything-you-need-to-know-about-fair-lending-regression-analysis) · [Premier Insights — stats ≠ disparate impact](https://www.premierinsights.com/blog/part-2-stats-in-fair-lending-why-statistical-evidence-doesnt-always-mean-disparate-impact) · [Ayres — included-variable bias](https://www.law.upenn.edu/live/files/1138-ayresincludedvariablebiaspdf)

### Anachronistic agency list — still verify-first
The 2009 procedures predate CFPB (2011) and were authored when OTS still existed (abolished 2011). Any retained "[5 agencies]" phrasing → soften to "the federal financial regulators." (Low-risk; include if convenient.)
